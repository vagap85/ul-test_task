# Разбор ошибок в коде 1С (подробно)

## Ошибка 1: МЕЖДУ
**Было:** `&СуммаВес МЕЖДУ ЦеныДоставки.ВесОт И ЦеныДоставки.ВесДо`

**Проблема:** Если интервалы [0,10] и [10,20], вес 10 попадёт в оба.

**Исправление:** `ЦеныДоставки.ВесОт < &СуммаВес И &СуммаВес <= ЦеныДоставки.ВесДо`

## Ошибка 2: ТекущаяДата()
**Было:** `Запрос.УстановитьПараметр("ДатаЗаказа", ТекущаяДата());`

**Проблема:** Игнорирует переданную дату заказа.

**Исправление:** `Запрос.УстановитьПараметр("ДатаЗаказа", ДатаЗаказа);`

## Ошибка 3: НайтиПоРеквизиту()
**Было:** `СправочникКлиенты.НайтиПоРеквизиту(...)`

**Проблема:** Если не найдёт — будет ошибка выполнения.

**Исправление:** 

Клиент = СправочникКлиенты.НайтиПоРеквизиту("ИНН", ИНН);
Если Клиент = Неопределено Тогда
Возврат 0;
КонецЕсли;


## Ошибка 4: Нет сортировки по ВесОт
**Проблема:** При одинаковом периоде запрос вернёт произвольную запись.

**Исправление:** Добавить в `УПОРЯДОЧИТЬ ПО` поле `ВесОт УБЫВ`

## Ошибка 5: Нет ПЕРВЫЕ 1
**Проблема:** Запрос возвращает все подходящие строки, хотя нужна только первая.

**Исправление:** Добавить `ВЫБРАТЬ ПЕРВЫЕ 1`

## Ошибка 6: Возврат 0
**Проблема:** 0 — валидная стоимость (бесплатная доставка). Нельзя отличить от "цены нет".

**Исправление:** Возвращать `Неопределено` или `NULL`

## Ошибка 7: Пустая табличная часть
**Проблема:** Если в заказе нет товаров, СуммаВес = 0, но это может быть ошибкой.

**Исправление:** 

Если СуммаВес = 0 Тогда
Возврат 0; // или Неопределено
КонецЕсли;
