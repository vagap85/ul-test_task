# Блок 4: API Testing (/api/v2/orders)

**Гайдлайны:** Google + Microsoft (ссылки в ТЗ)

## Чек-лист

### GET /orders

| Проверка | Ожидаемый код |
|----------|---------------|
| Всё ок, авторизован | 200 |
| Нет токена | 401 |
| Токен есть, но прав нет | 403 |
| Пагинация (page=2, limit=10) | 200 + Link, X-Total-Count |
| БД упала | 503 |

### POST /orders

| Проверка | Ожидаемый код |
|----------|---------------|
| Всё ок | 201 + Location |
| JSON сломан | 400 |
| Нет обязательного поля | 422 |
| Двойной запрос | 409 |
| Товара нет | 404 |

### GET /orders/{id}

| Проверка | Ожидаемый код |
|----------|---------------|
| Заказ есть | 200 |
| Заказа нет | 404 |
| Заказ чужой | 403 |

### PUT /orders/{id}

| Проверка | Ожидаемый код |
|----------|---------------|
| Ок | 200 |
| If-Match не совпал | 412 |
| Заказ удалён | 404 |

### DELETE /orders/{id}

| Проверка | Ожидаемый код |
|----------|---------------|
| Ок | 204 |
| Нет прав | 403 |
| Уже удалён | 404 |

## Дополнительно

| Проверка | Ожидаемый код |
|----------|---------------|
| Rate limit | 429 |
| Неверный метод | 405 |
| Content-Type не JSON | 415 |
